= {RedHatEdge} overview

The {RedHatEdge} provides streamlined management of edge devices and applications through a declarative approach. 
By defining the required state of your edge devices, which includes your operating system versions, host configurations, and application deployments, the {RedHatEdge} automatically implements and maintains these configurations across your entire device fleet.

== Integration capabilities

The {RedHatEdge} aims to offer the following:

* Support for external authentication providers such as Kubernetes and OpenShift Authentication API
// Keycloak, OIDC have been implemented but are not well tested

* Flexible authorization options such as Kubernetes RBAC
* Certificate management integration (built-in CA, Kubernetes Cert-Manager)
* GitOps workflow support for configuration management
